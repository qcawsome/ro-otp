# ro-otp
a little program helps ro user type otp easily


# 使用方式

下載 ro-otp-1.0.exe 到任意資料夾，設定羅技巨集

選擇鍵盤 -> 系統 -> 新增應用程式 -> 選擇剛剛下載的 ro-otp-1.0.exe -> 新增引數 輸入OTP的KEY

把增加的系統拉到要觸發的按鍵 -> 測試是否會自動輸入OTP
